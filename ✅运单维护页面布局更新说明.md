## 🎯 设计特点

### PageHeader 组件特性

```typescript
interface PageHeaderProps {
  title: string;           // 页面标题
  description: string;     // 页面描述
  icon: LucideIcon;       // 图标组件
  iconColor?: string;     // 图标颜色（可选）
  children?: ReactNode;   // 额外内容（可选）
}
```

### 布局特性

1. **Sticky Header**
   - 页面标题固定在顶部
   - 滚动时保持可见
   - `sticky top-4 z-10`

2. **响应式设计**
   - 移动端自适应
   - `flex-col md:flex-row`
   - 合理的间距和边距

3. **统一样式**
   - 白色背景 + 阴影
   - 圆角边框
   - 深色模式支持
