## 📊 "我司待开票总额" 含义详解

### 💡 核心概念
**"我司待开票总额"** 是指您的公司需要向合作方/客户开具发票的总金额。

### 🔍 具体含义
1. **业务背景**：运输业务完成后，您的公司需要向合作方开具发票
2. **计算逻辑**：`合作方应付金额 - 已开票金额 = 待开票金额`
3. **业务意义**：这是您公司尚未开具发票的应收款项

### 📈 财务流程
```
运输业务完成 → 生成运单 → 计算应付金额 → 开具发票 → 收款
                    ↓
              待开票总额（此阶段）
```

### 🎯 业务价值
- **现金流管理**：了解需要开具发票的金额，便于资金规划
- **税务合规**：确保及时开具发票，满足税务要求
- **客户关系**：及时开票有助于维护客户关系
- **财务控制**：监控开票进度，避免遗漏
