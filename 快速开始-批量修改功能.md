# 快速开始 - 批量修改功能

## 🚀 3步部署

### 步骤1：执行SQL（Supabase Dashboard）
```
1. 打开 Supabase Dashboard → SQL Editor
2. 执行文件：合作链路修改-使用正确重算逻辑.sql
3. 执行文件：supabase/migrations/20251025_add_batch_modify_functions.sql
```

### 步骤2：刷新浏览器
```
按 Ctrl + Shift + R
```

### 步骤3：开始使用
```
进入付款审核页面 → 选择运单 → 点击批量操作按钮
```

---

## 📋 功能预览

### 批量修改应收
- 显示运单列表：编号、日期、司机、原金额
- 每条运单独立输入框
- 可设置不同金额

### 批量修改链路
- 选择新的合作链路
- 自动重新计算成本
- 必须同一项目
