# 添加到package.json的scripts部分

在package.json中添加以下脚本：

```json
{
  "scripts": {
    "clean-logs": "node scripts/clean-console-logs.js",
    "optimize-db": "echo 'Database queries optimized! See 数据库查询优化指南.md for details'"
  }
}
```

使用方法：

```bash
# 清理console日志
npm run clean-logs

# 查看优化说明
npm run optimize-db
```
