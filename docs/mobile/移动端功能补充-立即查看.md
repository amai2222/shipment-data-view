## 🆕 新增的6个功能

### 1. 📍 地点管理（增强版）
**位置**：信息维护 → 地点管理（增强版）  
**功能**：GPS转换、批量导入、高级筛选

### 2. 🌲 货主层级管理
**位置**：信息维护 → 货主层级管理  
**功能**：管理总公司/子公司关系

### 3. 📄 开票申请单管理
**位置**：财务管理 → 开票申请单管理  
**功能**：查看管理所有开票申请

### 4. 🚚 运单维护
**位置**：数据维护（新分组）→ 运单维护  
**功能**：批量修改运单数据

### 5. 🚚⚡ 运单维护（增强版）
**位置**：数据维护 → 运单维护（增强版）  
**功能**：更强大的批量编辑

### 6. 🛡️ 权限配置
**位置**：设置 → 权限配置  
**功能**：配置用户权限
