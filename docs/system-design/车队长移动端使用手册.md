**角色**：车队长（fleet_manager）

## 🎯 车队长核心功能

### 1️⃣ 审核管理

**费用申请审核**：
- 查看司机提交的所有费用申请
- 审核通过/驳回
- 填写审批意见
- 批量审批（PC端）

**换车申请审批**：
- 查看司机的换车申请
- 审批通过后自动更新车辆关联
- 查看申请原因和历史

### 2️⃣ 车辆管理

**车辆档案**：
- 查看所有车辆状态
- 编辑车辆信息
- 上传/更新证件照片
- 记录维保信息

**司机-车辆分配**：
- 为司机分配主车
- 设置备用车
- 批准换车申请

**证件管理**：
- 查看证件到期提醒
- 更新证件照片
- 记录到期日期

### 3️⃣ 财务管理

**月度收入录入**（待开发）：
- 录入车辆月度运费收入
- 关联项目
- 自动生成流水

**收支流水**（待开发）：
- 查看车辆收支明细
- 按月份筛选
- 核对账目

## ⚠️ 权限范围

### ✅ 车队长可以：

1. **查看全部数据**
   - 所有司机的费用申请
   - 所有车辆信息
   - 所有司机档案
   - 所有运单记录

2. **审批管理**
   - 审批费用申请
   - 审批换车申请
   - 填写审批意见

3. **档案管理**
   - 编辑车辆信息
   - 编辑司机信息
   - 分配车辆
   - 配置项目线路

4. **财务操作**
   - 录入月度收入
   - 查看收支流水
   - 确认付款

### ❌ 车队长不能：

1. **系统设置**
   - 不能管理用户账号（仅 admin）
   - 不能修改系统权限

2. **跨系统操作**
   - 不能看外部司机的合同管理数据
   - 只能管理内部车辆系统

## 🔄 典型工作流程

### 每日早晨（8:00）
```
1. 打开手机，登录系统
2. 查看工作台首页
3. 看到待办提醒：
   - 🔴 3个费用申请待审核
   - 🟡 2个证件即将到期
4. 点击"费用审核"
5. 逐个审批费用申请
6. 通过的申请点击"确认付款"
7. 查看证件到期提醒
8. 联系司机更新证件
```

### 司机申请换车时
```
1. 收到待办提醒：换车申请待审批
2. 点击进入审批页面
3. 查看申请信息：
   - 司机：王师傅
   - 当前车：云F97310
   - 申请换成：云F88520
   - 原因：原车需要大修
4. 审核：
   - 查看车辆状态
   - 确认可用性
   - 填写审批意见
   - 点击"通过"
5. 系统自动：
   - 更新车辆关联
   - 王师傅主车变为云F88520
   - 通知司机
```

### 月底收入录入（待开发）
```
1. 打开"收入录入"
2. 选择车辆：云F97310
3. 选择月份：2025年11月
4. 输入运费收入：¥19,692
5. 选择收入来源：铁路配送项目
6. 保存
7. 系统自动生成流水记录
8. 更新车辆月度核算表
```
