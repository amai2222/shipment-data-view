## ⚠️ 关键注意事项

### 1. 姓名必须完全一致
```
✅ 正确：姓名填写 "王师傅"
❌ 错误：姓名填写 "王师傅-司机" 或 "王 师傅"

原因：系统通过 profiles.full_name = internal_drivers.name 关联
```

### 2. 角色选择
```
确保角色下拉框中能看到：
  - 车队长 ⭐
  - 司机 ⭐

如果看不到，说明步骤3脚本没执行成功
```

### 3. 创建顺序
```
建议先创建车队长，再创建司机
因为车队长需要审批司机的申请
```
